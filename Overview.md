### 第一部分：游戏总览
*   **基础定调**：游戏名为《不公正的哈尼》。
*   **品类与体量**：这是一款俯视角的 2D 动作角色扮演游戏（ARPG）。在底层架构上，它将采用 Godot 开源引擎进行开发，并采用纯 2D 的表现形式，这与《咩咩启示录》的 2.5D 透视有着本质区别。游戏的整体流程时长大约为 2 小时。

### 第二部分：核心要素与游戏流程
*   **三大核心要素**：
    *   **收集**：游戏内包含种类丰富的卡牌与货币系统。
    *   **探索**：玩家将在多联通、多层次的地图中进行游玩。玩家在游戏里的核心循环是一边探索地图、一边与对手战斗，同时一边收集卡牌。
    *   **战斗**：游戏主打高难度、重压力的对战体验。
*   **双层空间流程**：
    *   **前场迷宫**：这是一个立体连通的空间，玩家在此收集资源、规避陷阱，同时感受游戏的氛围与剧情。在这里的主要行为是击败杂兵以收集资源。收集到的资源用于增强数值（如经验、货币）以及抽取卡牌（如技能、道具）。
    *   **后场舞台**：这是一个独立的房间环境，专门用于挑战 BOSS。后场舞台的特点是激战、高压力、需要玩家反复挑战，并伴有华丽的演出效果。
*   **推进目标**：玩家需要不断探索地图，通过战斗获取强力卡牌，再利用这些卡牌去挑战后续的战斗，最终目标是通关并收集完所有的卡牌。卡牌的获取途径分为首次战胜获得的普通卡牌，以及达成限制条件战胜后获得的稀有卡牌。

### 第三部分：高压战斗系统
*   **战斗节奏**：游戏的核心玩法围绕着高强度的动作打斗展开。
*   **攻击方式**：在面对敌人高压攻击的间隙，玩家需要使用普通攻击来尽量削减敌人的血量。
*   **卡牌技能**：玩家必须找到合适的时机，释放具有一次性消耗性质的强力卡牌技能来造成大量伤害。此外，系统允许玩家额外选择 1 张名为“作弊卡牌”的特殊卡牌进行反复使用。

### 第四部分：双轨收集与构筑系统
游戏内的所有操作均依靠两种核心收集品来实现：
*   **属性块**：这是一种基础资源，既代表了角色的攻防属性，也是用于抽卡的货币。玩家能力的数值并非固定，而是可以随着属性块的布置情况进行修改。
*   **卡牌**：卡牌的功能极为庞杂。除了可以在战斗中布置使用外，它还承载了人物图鉴的属性加成、触发特殊剧情、作为战斗技能、提供有限存档功能、记录成就奖章、作为携带或消耗物品，甚至用于更换武器等多种用途。
*   **构筑逻辑**：玩家将这些收集品布置在界面上的位置和数量，将直接决定自身最终的面板数值（如体力、物理攻击、法术防御等）。

### 第五部分：不公平的系统数值
*   **系统误导**：在游戏前期，玩家的装备系统是被系统刻意误导的，只能极其有限地使用极少的装备槽位。
*   **敌我差距**：与此同时，敌人的装备却具备完整的功能，这导致玩家与敌人在装备数值上存在巨大差距。
*   **认知反转**：直到游戏中期，当玩家看到敌人的装备界面时，才会发现自己的思路受到了限制。原来从游戏一开始，玩家就可以仿照敌人的布置方式，使用完整的装备系统。

### 第六部分：物理 UI 作弊机制
玩家在发觉与对手的实力差距后，可以通过物理交互 UI 界面的方式进行“作弊”：
*   **隐藏槽位**：通过多次点击界面的特定位置，可以“打开上盖”，从而使用里面隐藏的 3 个槽位。
*   **无视规则**：只要将武器、技能、加成或道具卡牌布置在这 3 个隐藏槽位中，就可以无视原有条件，使其变成非一次性且可无限次使用的卡牌。
*   **强行布置与翻倍**：对于 UI 周边原本没有挖槽的空白位置，玩家通过反复拖拽属性块也可以强行布置上去。如果全部布置满，角色的属性数值可以实现翻倍。
*   **替换计算器**：游戏中存在不同的“计算器”，且各自的计算公式不同。如果玩家将计算器更换为“哈尼的计算器”，系统将直接取消数值的最大值限制。

### 第七部分：动态垄断经济学
游戏内的资源分为“还在流通”与“已被对手垄断”两种状态：
*   **属性垄断**：如果某个团体（敌方或玩家方）持有了一定数量的某种属性块，就可以宣告对该属性进行垄断。一旦被垄断，该属性块将不再发行，其他团体只能通过抢夺来获取。
*   **卡牌垄断与定价权**：如果团体消耗同一张卡牌达到一定次数，同样可以宣告垄断。垄断后，该团体将拥有这张卡的“定价权”。当其他团体想要使用这张卡牌时，就必须支付高额的代价（例如从消耗 20 点属性变为消耗 100 点属性）。
*   **动态变化**：这种垄断从来都不是永久性的。随着游戏进度的推进，玩家日后也可以凭借自身实力去垄断这些资源。

### 第八部分：从不公平到作弊的闭环
游戏刻意营造了三种不对等的体验阶段：
*   **不公平（敌人强）**：系统规则虽然没有说谎，但故意没有对玩家讲全。强力的资源被老玩家（敌人）们所垄断，市面上流通的大多只是基础资源。
*   **公平（大家一样强）**：玩家通过战斗察觉到实力差距，随后开始模仿。因为玩家与敌人其实共用同一套底层系统，玩家可以通过仿照敌人的配置来大幅提升自身能力。
*   **不公平（原来我更强/作弊）**：除了玩家主动的 UI 作弊，系统还提供了一种未知的优待：负责抽卡的 NPC 会出于某种原因偷偷帮玩家作弊。当玩家达成特定条件后，NPC 会无视规则，让玩家直接抽到强力卡牌。

### 第九部分：特色美术与克制叙事
*   **视觉呈现**：游戏具有极具特色的场景美术，注重整体的“用色美”。同时，游戏内的特色卡牌也拥有极高辨识度的美术风格。
*   **叙事基调**：这是一个以逻辑严谨、对白克制为核心卖点的群像型故事。
*   **冰山理论与演出**：开发团队在规划完 100% 的完整剧情后，在设计上仅有限地给观众展示 10% 的内容。游戏主要追求营造整体的氛围效果，通过“勾引”的方式引导玩家自行探索剧情拼图。在保持台词简洁的前提下，尽量通过关键事件和战斗风格的演出来塑造角色并推进剧情。
*   **避免吃书**：为了避免生硬的大段对白，游戏遵循不同角色的动机和性格，以“直接描写结论”的形式推进故事。虽然观众无法直接看到故事全貌，但底层逻辑依然严格按照既定大纲展开，目的是全力避免“吃书”以及角色前后行为逻辑出现矛盾的情况。
*   **玩家的边缘立场**：在整体故事的因果链条中，玩家处于一个“无关大局”且“被人利用”的边缘立场上。因此在绝大多数时候，玩家只能面对眼前生存与战斗的困境，只有在极其偶然的情况下才能窥见故事的丝丝真相。这种设计使得即便是不愿意深究剧情的玩家，也能毫无负担地充分享受战斗的乐趣。
